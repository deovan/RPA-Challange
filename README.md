
# INSTALAÇÂO

  - Instalar o [node.js](https://nodejs.org/en/)
  - Instalar o [git bash](https://git-scm.com/downloads)
  
  Após ter feito a instação, executar o comando abaixo na pasta onde está o robô:

  `
  npm i
  `
